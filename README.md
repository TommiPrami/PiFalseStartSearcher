# PiFalseStartSearcher
Triny tool to find longest "False restart" of the Pi. JUst occurred to me that what is to longest false restart,and manually going trough Trillion decimals seemed a bit tounting task..

To use this you will need a text file with Pi, with "Ansi encoding". File must contain only the numbers and the '.' char.

This seems to be pretty fast, there would be room for optimization for sure.
